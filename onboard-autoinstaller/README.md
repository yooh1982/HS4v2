# 스마트십 플랫폼 자동 설치기 (Onboard Auto Installer)

Ubuntu 서버에 스마트십 플랫폼을 **Excel/CSV 기반 입력**과 **정해진 규칙**으로 자동 설치하고,  
작업 로그와 **결과 보고서**를 생성하는 도구입니다.

---

## 목적

- **입력**: 설치 정보(Excel/CSV) + 타깃 호스트 정보(Excel/CSV)
- **동작**: 규칙에 따라 각 호스트에 설치 수행, 문제 발생 시 해결 시도 및 로깅
- **출력**: 전체 작업 로그 파일 + 설치 결과 보고서

## 디렉터리 구조

```
onboard-autoinstaller/
├── README.md
├── docs/
│   └── ARCHITECTURE.md    # 아키텍처 문서
├── config/                # 설치 규칙
├── input/                 # 설치 정보·타깃 호스트 (CSV/Excel)
├── scripts/               # 설치·롤백 스크립트
├── templates/             # 설정 템플릿 (필요 시)
├── logs/                  # 실행 로그 (실행 시 생성)
└── reports/               # 작업 결과 보고서 (실행 후 생성)
```

---

## 1차 범위

- CSV/Excel 기반 설치 정보·호스트 정보 입력
- 규칙 기반 원격 설치 (SSH)
- 파일 로깅 및 작업 결과 보고서 생성
- (2차) 설치 후 점검은 별도 단계에서 추가 예정

---

## 진행 순서

1. **현재**: 컨셉 및 아키텍처 문서 작성 완료 → **피드백 수렴**
2. **다음**: 피드백 반영 후 설치 규칙·입력 포맷·스크립트 구체화 및 실제 설치 진행

피드백 주시면 그에 맞춰 설계를 수정하고, 이어서 설치 단계를 진행하겠습니다.
