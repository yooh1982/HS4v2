# OBDP Install Check

Installer를 통해 설치한 플랫폼이 정상적으로 설치되었는지 **초기 설치 상태**를 확인하는 도구입니다.

## 목적

- 설치 스크립트(installer) 실행 후 플랫폼의 초기 설치 상태 검증
- 필수 구성요소·경로·설정 등이 기대한 대로 존재하는지 확인

## 디렉터리 구조

```
OBDP_install_check/
├── README.md
├── scripts/       # 설치 검증 스크립트 (추가 예정)
├── config/        # 검증 규칙·기준 설정 (선택)
└── docs/          # 사용법·체크 항목 문서 (선택)
```
